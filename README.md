# TabSystem

## How To Use? (example)

```javascript
<div tab="system">

    <div tab_target="tab1">
        Show [1]
    </div>

    <div tab_target="tab2">
        Show [2]
    </div>

    <div tab_area>

    </div>

    <div tab_data="tab1">
        I'M DATA 1
    </div>

    <div tab_data="tab2">
        I'M DATA 2
    </div>

</div>
```
